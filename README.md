# Liquidity Advisor Calculator

A React-based calculator for liquidity and profit analysis.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment on Koyeb

1. Push this code to a GitHub repository.
2. In Koyeb, create a new static site app.
3. Connect your GitHub repository.
4. Set the build command to: `npm run build`
5. Set the output directory to: `dist`
6. Deploy!
